sphinx-icon
===========

.. toctree::
   :hidden:

   quickstart
   contribute

The icon extension allows you to embed fontawesome icons as inline roles in a sphinx documentation.

.. code-block:: rst

    I'm a folder icon: :icon:`fa fa-folder`.

I'm a folder icon: :icon:`fa fa-folder`.
